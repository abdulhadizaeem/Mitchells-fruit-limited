# Mitchell's Fruit Farms — React + JavaScript Admin Portal

This is the React + JavaScript frontend dashboard for Mitchell's Fruit Farms. It serves as the central control panel for monitoring AI voice agent call logs, reviewing Clover POS drafts, and managing settings.

## 🛠️ Tech Stack & Setup
- **Framework**: React 19 + JavaScript (JS/JSX)
- **Bundler**: Vite
- **Styling**: Tailwind CSS v4
- **Routing**: React Router DOM v7
- **HTTP Client**: Axios with JWT interception

## 🚀 Running Locally

1. Install dependencies:
   ```bash
   npm install
   ```

2. Configure environment:
   Create a `.env` file in the `frontend` root:
   ```ini
   VITE_BASE_URL=http://localhost:8000/api
   ```

3. Start development server:
   ```bash
   npm run dev
   ```

4. Production Build:
   ```bash
   npm run build
   ```
